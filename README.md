# API Data Retrieval, Processing & SQLite Storage Project

## 📌 Overview
This project demonstrates:
- Fetching data from an API
- Storing data in SQLite database
- Data processing & visualization
- Importing CSV data into database

---

## 🚀 Features

### 1. API Data Retrieval & Storage
- Fetches book data from an API
- Stores data in SQLite (`books` table)
- Displays stored records

### 2. Data Processing & Visualization
- Fetches student scores
- Calculates average score
- Displays bar chart using matplotlib

### 3. CSV to Database
- Reads user data from CSV file
- Inserts into SQLite (`users` table)

---

## 🛠️ Technologies Used
- Python
- SQLite
- Pandas
- Matplotlib
- Requests
